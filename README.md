move3d-openrave-plugin
==============

Depends on move3d and OpenRave

http://www.openrobots.org/wiki/move3d

http://openrave.org/docs/latest_stable/


Install:

    cd ormove3d
    mkdir build
    cd build
    cmake ..
    make install
    
Run:
    
    cd examples
    python test_move3d_birrt_stones.py
